## Description of the game

The game involves you being a ghost and wanting to explore the forest you are in and see what happens.

## How to play

- **WSDA KEYS**: Walk.
- **Spacebar**: jump.
- **F KEY**: Shoot
- **Esc**: Exit the game.

## Debug keys

- F1 : Start from the first level
- F5 : Save the current game state
- F6 : Load the previous state
- F9 : Show collisions/Pathfinding routes
- F10 : God mod
- F11 : Enable/Disable FPS cap to 30

## Game Objective:
DODGE TRAPS AND ENEMYS, YOU CAN ALSO KILL THEM, JUMP OVER PITS, EXPLORE THE MAP!!!
