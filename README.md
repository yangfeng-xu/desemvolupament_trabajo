## How to play

- **WSDA KEYS**: Walk.
- **Spacebar**: jump.
- **Esc**: Exit the game.

## Debug keys

- F1 : Show collisions
- F10 : God mod

### Game Objective:
DODGE TRAPS, JUMP OVER PITS, EXPLORE THE MAP!!!

## Features Implemented

- One level.
- Movement system.
- Jumping system.
- Background music during gameplay.
